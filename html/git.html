Git

  • git add - добавление файлов в папке.
  • git status - проверка статуса репозитория, показывает какие файлы попали в проект, какие находятся в индексе и какие сохраненные файлы изменены.
  • git commit - добавление файлов в репозиторий.
  • git log -просмотр журнала коммита.
  • git show - просмотр коммита.
  • git diff- просмотр изменений до коммита.
  • git push- отправка изменений в удаленный репозиторий.
  • git pull- получение изменений из удаленного репозитория.
